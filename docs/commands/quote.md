# /quote

## Syntax

    /quote <raw command>

## Description

Send a raw IRC command directly to the server. This is useful for
commands not yet implemented, testing, or advanced protocol interaction.

## Aliases

/raw

## Examples

    /quote VERSION
    /raw LUSERS
    /quote PRIVMSG #channel :hello
