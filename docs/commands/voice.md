# /voice

## Syntax

    /voice <nick> [nick2 ...]

## Description

Give voice status (+v) to one or more users in the current channel.
Voiced users can speak in moderated (+m) channels. You must be a
channel operator to use this command.

## Examples

    /voice user1
    /voice user1 user2
