# /deop

## Syntax

    /deop <nick> [nick2 ...]

## Description

Remove channel operator status (-o) from one or more users in the current
channel. You must be a channel operator to use this command.

## Examples

    /deop troublemaker
    /deop user1 user2

## See Also

/op, /devoice, /mode
