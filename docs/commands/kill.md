# /kill

## Syntax

    /kill <nick> [reason]

## Description

Forcefully disconnect a user from the IRC network. This is an operator-only
command. If no reason is given, the target's nick is used as the reason.

## Examples

    /kill spammer Spamming is not allowed
    /kill baduser

## See Also

/oper, /kick, /ban
