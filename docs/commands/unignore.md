# /unignore

## Syntax

    /unignore <number|mask>

## Description

Remove an ignore rule by its number (from `/ignore` list) or
by the exact mask pattern.

## Examples

    /unignore 1
    /unignore troll
    /unignore *!*@bad.host.com
