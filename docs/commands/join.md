# /join

## Syntax

    /join <channel> [key]

## Description

Join an IRC channel. A `#` prefix is auto-added if missing.
Optionally provide a channel key for key-protected channels.

## Examples

    /join #linux
    /join linux
    /join #secret mysecretkey

## See Also

/part, /close
