# /op

## Syntax

    /op <nick> [nick2 ...]

## Description

Give channel operator status (+o) to one or more users in the current channel.
You must be a channel operator to use this command.

## Examples

    /op friend
    /op user1 user2 user3

/deop, /voice, /mode
