# /links

## Syntax

    /links [remote] [mask]

## Description

List the servers linked to the IRC network. Optionally provide a remote server to query from, and a mask pattern to filter the results.

## Examples

    /links
    /links *.libera.chat
    /links irc.libera.chat *.eu
