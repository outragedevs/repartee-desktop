# /oper

## Syntax

    /oper <name> <password>

## Description

Authenticate as an IRC operator. Requires valid operator credentials
configured on the server. Once authenticated, you gain access to
operator commands like /kill and /wallops.

## Examples

    /oper admin secretpass

## See Also

/kill, /wallops
