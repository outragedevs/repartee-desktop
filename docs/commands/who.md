# /who

## Syntax

    /who <target>

## Description

Send a WHO query to the server. The target can be a channel name, a nick, or a mask pattern.

Returns information about matching users including nick, username, hostname, server, and real name.

## Examples

    /who #linux
    /who nick
    /who *.fi
